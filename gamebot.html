Let me introduce Project.

A Reinforcement learning based Gamebot works on Deep Q learning model principles.

Softwares used

  • Spyder
  • Python
  • OpenAI
  • Atom
  • TensorFlow
  • Keras
  • Opencv

Ideas Behind Project

  • Works on Reinforcement Learning
  • Learns slowly by observing the environment and observing rewards of actions
  • A Convolution Neural Network is used to Process each stage of screen
  • A model is built based on the best actions to maximize rewards
  • Finally a player predicts from the trained model and do the action accordingly.
