About

Let me introduce Project.

AgroQuad as name is the combination of two things Agriculture and Quadcopter. You must be wondering how the Quadcopter is related to Agriculture, but this combination can increase the productivity and growth of Agriculture sector marginally. In this a Quadcopter will be used to survey the area of the farm and during this survey it will click certain images of farm. Later these images are proccessed by one of the best machine learning algorithms so that we can predict the health and diseases in the farm to tell farmers neccssary preventions. This will decrease the loss due to improper crop and diseases. The path travelled by Quadcopter is also optimized in order to save energy and save time.

Ideas Behind Project

  • Using the dataset of previous labelled images we can predict the label of new image
  • We can control the Quadcopter using Android by sending commands to Raspberry Pi over the WiFi
  • Multiwii is device which is used to control the 4 axis motors like Rc cars, Quadcopter
  • Genetic Algorihtms are used to find optimized solution from the using mutation and crossover
  • A path over an area can be find using depth first search algorithm A*
